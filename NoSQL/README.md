- Ce que signifie NoSQL
- La différence entre SQL et NoSQL
- Ce qu'est ACID
- Ce qu'est un stockage de documents
- Les types de NoSQL
- Les avantages d'une base de données NoSQL
- Comment interroger des informations à partir d'une base de données NoSQL
- Comment insérer/mettre à jour/supprimer des informations dans une base de données NoSQL
- Comment utiliser MongoDB

### Exigences
#### Fichiers de commande MongoDB
- Tous vos fichiers seront interprétés/compilés sur Ubuntu 20.04 LTS en utilisant MongoDB (version 4.4)
- Tous vos fichiers doivent se terminer par une nouvelle ligne
- La première ligne de tous vos fichiers doit être un commentaire : `// mon commentaire`
- Un fichier `README.md`, à la racine du dossier du projet, est obligatoire
- La longueur de vos fichiers sera testée avec `wc`

#### Scripts Python
- Tous vos fichiers seront interprétés/compilés sur Ubuntu 20.04 LTS en utilisant python3 (version 3.9) et PyMongo (version 4.8.0)
- Tous vos fichiers doivent se terminer par une nouvelle ligne
- La première ligne de tous vos fichiers doit être exactement `#!/usr/bin/env python3`
- Un fichier `README.md`, à la racine du dossier du projet, est obligatoire
- Votre code doit utiliser le style `pycodestyle` (version 2.5.*)
- La longueur de vos fichiers sera testée avec `wc`
- Tous vos modules doivent avoir une documentation (`python3 -c 'print(__import__("my_module").__doc__)'`)
- Toutes vos fonctions doivent avoir une documentation (`python3 -c 'print(__import__("my_module").my_function.__doc__)'`)
- Votre code ne doit pas être exécuté lors de l'importation (en utilisant `if __name__ == "__main__":`)

### Synthèse de l'Introduction et du Manuel
Ce projet vise à vous familiariser avec les bases de données NoSQL, en particulier MongoDB. Vous apprendrez les concepts fondamentaux de NoSQL, les différences entre SQL et NoSQL, et les avantages de l'utilisation d'une base de données NoSQL. Vous serez également formé à l'utilisation de MongoDB, y compris l'insertion, la mise à jour, la suppression et l'interrogation de données. Le projet comprend des ressources pour vous aider à comprendre ces concepts et des exigences strictes pour vos fichiers de commande MongoDB et scripts Python. Enfin, vous devrez installer MongoDB 4.4 sur Ubuntu 20.04 et suivre les meilleures pratiques de codage.
